Desafío Suricata
*******************

Se ocupó Bootstrap 4.5 para la creación del navegador, a éste le agregamos un contenedor que ayuda a
posicionar el logo de Suricata. 
Además, ocupamos el componente "card", en conjunto con la opción "card-deck" para que éstas no se desordenen dentro de la web, al momento de añadir o quitar texto. Cada una de estas tarjetas, ocupa columnas de acuerdo a la documentación de Bootstrap, lo que las hace responsivas.
Para finalizar, un footer que contiene iconos y el logo de la web, han sido centrados de la forma en que la documentación lo sugiere.
